HTML5 Boilerplate + Harp

This is an example boilerplate using HTML5 Boilerplate with Bootstap and Harp. Harp can pre-compile LESS so we use the LESS version of Twitter Bootstrap.

Learn more »

This is a Port of HTML5 Boilerplate to Harp and makes use of many of Harps features

  • Templates with EJS and Jade
  • Partials and Layouts
  • Flexible Meta Data
  • Custom 404 Page
  • Pre-compiling LESS
